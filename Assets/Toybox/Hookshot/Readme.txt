# Hookshot Prefab v1

A functional hookshot for use in VRChat

# Layers

UserLayer22: Respawn boundary
UserLayer23: Hookshot Surface
UserLayer24: Internal Use

# Animator Variables

IsMuted:
- Disable sound effects

IsPvp:
- Allow the local player to be grabbed by
  the hookshot

# Notes

Place a large trigger volume on
UserLayer23 at the bottom of the world
above the standard respawn boundary to
ensure the hookshot can respawn properly.
